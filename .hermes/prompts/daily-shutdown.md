# Prompt: Daily Shutdown

## Context
End the development session and preserve state.

## Shutdown Sequence
1. Run full test suite to ensure nothing is broken
2. Commit any work-in-progress with descriptive messages
3. Update project memory with session outcomes
4. Document any discovered issues or technical debt
5. Prepare handoff notes for next session
6. Generate session summary report

## Output
- Commit summary
- Updated project memory
- Session report
- Tomorrow's recommended priorities
