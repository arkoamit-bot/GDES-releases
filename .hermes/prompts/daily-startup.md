# Prompt: Daily Startup

## Context
Begin a new development session for the GDES project.

## Startup Sequence
1. Check git status and recent changes
2. Review any open issues or TODOs
3. Run quality gates (pytest, ruff, mypy)
4. Check for pending migrations
5. Review project memory for context
6. Identify today's priorities
7. Suggest a work plan

## Output
- Repository status summary
- Quality gate results
- Recommended daily priorities
- Any blocking issues
