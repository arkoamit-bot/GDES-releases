# Prompt: Repository Review

## Context
You are reviewing the GDES repository for overall health, architecture, and readiness.

## Instructions
1. Inspect the directory structure and identify all Django applications
2. Review models.py files across all apps for domain model consistency
3. Check URL configurations for API consistency
4. Review serializer patterns across apps
5. Examine test coverage and quality
6. Check documentation currency
7. Identify any architectural anti-patterns
8. Report findings with severity levels

## Output Format
- Executive Summary
- Architecture Assessment (1-10 score)
- Findings categorized by severity (Critical/High/Medium/Low)
- Recommendations with priority ordering
- Risk assessment
