# Project Memory Index

## Files
- `PROJECT_MEMORY.md` — Core project memory (auto-updated)
- `SESSION_LOG.md` — Development session history
- `KNOWN_ISSUES.md` — Tracked issues and workarounds
- `DECISIONS.md` — Architecture and design decisions

## Update Schedule
- **PROJECT_MEMORY.md**: Updated daily and after major changes
- **SESSION_LOG.md**: Updated at session start/end
- **KNOWN_ISSUES.md**: Updated when issues discovered
- **DECISIONS.md**: Updated when decisions made

## Usage
- Always read PROJECT_MEMORY.md before starting work
- Record significant decisions in DECISIONS.md
- Track known issues for future reference
