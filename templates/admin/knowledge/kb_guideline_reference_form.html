Import guideline (reference)

Store a guideline (e.g. a KDIGO PDF) as a citable reference. The text is extracted and saved as a Guideline document under a Guideline source, so your rules can cite it (evidence link, chapter, quote) and you can read the source inside the app. This does not create or change any clinical rules — turning guideline prose into rules is a clinical authoring task done separately.
