## 2. Drug Classification

| Drug Class | Role | Evidence |
|---|---|---|
| Methylprednisolone | Pulse steroids for acute ABMR | KDIGO 1C |
| IVIG (Intravenous immunoglobulin) | Neutralise DSA, immunomodulation | KDIGO 1C |
| Plasma exchange (PLEX) | Remove circulating DSA | KDIGO 1C |
| Rituximab (anti-CD20) | Deplete B-cells (severe/refractory ABMR) | KDIGO 2C |
| Bortezomib (proteasome inhibitor) | Deplete plasma cells (refractory) | AST 2C |
| Carfilzomib (2nd-gen proteasome inhibitor) | Alternative proteasome inhibitor | AST 2D |
| Eculizumab (anti-C5) | Complement blockade (limited evidence) | AST 2D |
| Tacrolimus (CNI) | Baseline immunosuppression maintenance | KDIGO 1A |
| Mycophenolate mofetil (MMF) | Anti-proliferative maintenance | KDIGO 1A |
| Prednisone | Maintenance immunosuppression | KDIGO 1B |
| ACE inhibitors (Ramipril) | Renoprotection for proteinuria | KDIGO 1B |
| ARBs (Losartan) | Alternative renoprotection | KDIGO 1B |
| CCBs (Amlodipine) | BP management | KDIGO 1C |

## 3. Treatment Protocol by Phase

| Phase | Regimen | Duration |
|---|---|---|
| Acute ABMR induction | MP 500 mg IV x3 + IVIG 2 g/kg + PLEX x5 | 7-14 days |
| Severe/refractory | Add Rituximab 375 mg/m2 x1 | Day 1 |
| Refractory (persistent DSA) | Bortezomib 1.3 mg/m2 x4 | Days 1, 4, 8, 11 |
| Chronic ABMR | IVIG 2 g/kg monthly | 3-6 months |
| Maintenance | Tacrolimus target 5-8 + MMF 2g + Pred taper | Lifelong |
