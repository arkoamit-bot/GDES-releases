## 3. Treatment Protocol by Type
| TCMR Type | Regimen | Duration |
|---|---|---|
| Type I | MP 500 mg IV x3-5 + prednisone taper | 3-5 days pulse, 2-4 wk taper |
| Type II | MP 500 mg IV x3-5 + ATG 1.5 mg/kg x5-14 | 5-14 days ATG |
| Type III | ATG 1.5 mg/kg x7-14 days | 7-14 days |
| Steroid-resistant | ATG 1.5 mg/kg x7-14 days (regardless of type) | 7-14 days |
| ATG-resistant | Alemtuzumab 30 mg x1-2 (rescue) | Per protocol |
