## 2. Drug Classification

| Drug Class | Role | Evidence |
|---|---|---|
| INSTIs (Dolutegravir, Bictegravir) | ART backbone — suppress HIV viremia | DHHS 1A |
| NRTIs (TAF+FTC, ABC+3TC) | ART backbone — dual NRTI | DHHS 1A |
| ACE inhibitors (Ramipril) | Renoprotection — reduce proteinuria | KDIGO 1B |
| ARBs (Losartan) | Alternative renoprotection | KDIGO 1B |
| Corticosteroids (Prednisone) | Rapidly progressive collapsing GN | KDIGO 2D |
| CCBs (Amlodipine) | BP management | KDIGO 1C |
| Loop diuretics (Furosemide) | Edema management | KDIGO 1C |
| Statins (Atorvastatin) | CV risk reduction | KDIGO 1C |

## 3. Key Drugs

### 3.1 INSTIs (Integrase Strand Transfer Inhibitors)
- Dolutegravir 50 mg daily or Bictegravir 50 mg daily
- First-line ART for HIVAN — potent, well-tolerated, no nephrotoxicity
- Monitor: Cr (DTG increases Cr via OAT2 inhibition — benign 0.1-0.2 mg/dL rise)

### 3.2 NRTI Backbone
- TAF 25 mg + FTC 200 mg (preferred — no nephrotoxicity)
- Avoid TDF if eGFR <60 (tubular toxicity, Fanconi syndrome)
- ABC + 3TC alternative if HLA-B*5701 negative

### 3.3 RAASi
- Ramipril 2.5-10 mg daily. Start low, titrate.
- Monitor K+, Cr 2-4 wks after initiation.

### 3.4 Prednisone (Limited Role)
- 1 mg/kg/day x8 wks then taper over 6-12 weeks
- Caution: worsens HIV if CD4 low, IRIS risk
- Only for rapidly progressive collapsing GN despite ART

## 4. Drug Safety

| Drug | Key Side Effects | Monitoring |
|---|---|---|
| DTG | Insomnia, headache, weight gain, Cr increase | Cr (benign rise) |
| TAF | Low toxicity profile | eGFR |
| TDF | Fanconi, AKI, osteopenia | Phosphate, glucose, eGFR |
| Ramipril | Cough, hyperkalemia, AKI | K+, Cr |
| Prednisone | Hyperglycemia, infection, IRIS | Glucose, CD4 |

## 5. ART-Drug Interactions

| ART | Immunosuppressant | Interaction |
|---|---|---|
| DTG | None significant | Safe with all |
| TAF | None significant | Safe with all |
| Protease inhibitors (avoid) | Calcineurin inhibitors | Increased CNI levels |
| NNRTIs (EFV) | MMF | Possible decreased MMF |
