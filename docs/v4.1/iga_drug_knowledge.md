## 2. Drug Classification
| Drug Class | Role | Evidence |
|---|---|---|
| ACEi/ARB | First-line antiproteinuric | KDIGO 1B |
| SGLT2i (Dapagliflozin) | Add-on renoprotection | DAPA-CKD 1A |
| Methylprednisolone | High-risk / crescentic disease | KDIGO 2B |
| Mycophenolate Mofetil (MMF) | Alternative IS (Chinese studies) | KDIGO 2C |
| Rituximab (Anti-CD20) | Refractory disease (off-label) | KDIGO not recommended |
| Omega-3 PUFA (Fish Oil) | Adjunctive renoprotection | KDIGO 2D |
| Hydroxychloroquine | Adjunctive antiproteinuric (China) | KDIGO 2C |
| TRF-Budesonide (Nefecon) | Gut mucosa IS | NEFIGAN phase 2b |
| Icodextrin | PD for ESKD | Standard care |

## 3. Acute vs Chronic Management
| Phase | Intervention | Goal |
|---|---|---|
| Acute flare (macroscopic hematuria) | Supportive, hydration, avoid NSAIDs | Resolve, exclude crescentic |
| High-risk proteinuria >1g/d | ACEi/ARB + SGLT2i | <0.5g/d or 50% reduction |
| Crescentic disease | Pulse MP + MMF/Cyclo | Rescue renal function |
| Chronic CKD | Renoprotection + monitor | Delay ESKD |
| ESKD | RRT or transplant | Long-term survival |
