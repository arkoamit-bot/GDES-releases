## 2. Drug Classification
| Drug Class | Role | Evidence |
|---|---|---|
| CNI (Tacrolimus) | Baseline IS - maintain at lower trough | KDIGO 1B |
| Belatacept | CNI-free alternative IS | BENEFIT 1B |
| IVIG | DSA+ active TG - immunomodulation | KDIGO 2C |
| Rituximab | Refractory DSA+ TG | KDIGO 2C |
| Bortezomib | Refractory DSA (plasma cell targeted) | Off-label 2D |
| ACEi/ARB | First-line antiproteinuric | KDIGO 1B |
| SGLT2i (Dapagliflozin) | Add-on renoprotection | DAPA-CKD 1A |
| Loop diuretics | Oedema management | Standard care |
| Eculizumab | TMA component (off-label) | Case reports |
---
## 3. Acute vs Chronic Management
| Phase | Intervention | Goal |
|---|---|---|
| Early TG (cg1a, subclinical) | Monitor + IVIG single dose | Prevent progression |
| Active TG (cg1b-cg2, DSA+, eGFR declining) | IVIG + rituximab, CNI minimise | Stabilise function |
| Refractory TG (cg2-cg3, rising DSA) | Bortezomib, belatacept conversion | Slow graft loss |
| Advanced TG (cg3, eGFR <20) | Dialysis planning, palliation | Prepare for RRT |
| Retransplant | DSA desensitisation pre-transplant | Successful regraft |
