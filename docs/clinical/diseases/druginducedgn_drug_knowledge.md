## 2. Drug Classification

| Drug Class | Role | Evidence |
|---|---|---|
| Drug withdrawal | Cornerstone — discontinue offending agent | KDIGO 1C |
| ACE inhibitors (Ramipril) | Renoprotection for persistent proteinuria | KDIGO 1B |
| ARBs (Losartan) | Alternative renoprotection | KDIGO 1B |
| Prednisone | ICI-ATIN: 1 mg/kg tapered over 4-8 weeks | ASN/KDIGO 1B |
| Prednisone | Persistent NSAID-MCD >12 wks | KDIGO 2C |
| Prednisone | Collapsing FSGS (limited evidence) | KDIGO 2D |
| Valproate | Lithium alternative mood stabiliser | ERA/EDTA 2C |
| Hydroxychloroquine | Anti-TNF lupus (mild-moderate) | KDIGO 2C |
| MMF | Anti-TNF lupus (severe renal) | KDIGO 1B |
| MMF | ICI-collapsing GN (adjunctive) | ISN 2D |
| Loop diuretics (Furosemide) | Edema management | KDIGO 1C |
| CCBs (Amlodipine) | BP management | KDIGO 1C |
| Statins (Atorvastatin) | CV risk reduction | KDIGO 1C |
| Methadone/Buprenorphine | Heroin cessation therapy | ASAM 2023 |

## 3. Treatment-by-Drug Class

| Offending Drug | First-Line | Second-Line | Alternative |
|---|---|---|---|
| NSAIDs | Withdrawal | Prednisone if MCD >12 wks | RAASi |
| Lithium | Switch to valproate/atypical antipsychotic | RAASi | N/A |
| Pamidronate | Withdrawal | Prednisone (limited) | Switch to zoledronate |
| Interferons | Withdrawal | Prednisone (limited) | N/A |
| ICIs (ATIN) | Prednisone 1 mg/kg | ICI rechallenge after resolution | N/A |
| ICIs (collapsing GN) | Withdrawal + Prednisone | MMF | Dialysis |
| Anti-TNF | Withdrawal | HCQ/Prednisone | MMF |
| Heroin | Cessation program + RAASi | ESKD management | Transplant |
