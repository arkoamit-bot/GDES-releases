## 2. Drug Classification
| Drug Class | Role | Evidence |
|---|---|---|
| Tacrolimus (CNI) | Causative agent — reduce dose | KDIGO 1B |
| Cyclosporine (CNI) | Alternative CNI (less diabetogenic) | KDIGO 2C |
| Belatacept | CNI-free IS — preserves GFR | KDIGO 1B |
| Sirolimus/Everolimus (mTOR inhibitor) | Alternative to CNI — monitor proteinuria | KDIGO 2C |
| ACE inhibitors (Ramipril) | Renoprotection, proteinuria reduction | KDIGO 1B |
| ARBs (Losartan) | Alternative renoprotection | KDIGO 1B |
| Patiromer | Hyperkalaemia management | KDIGO 2C |
| Fludrocortisone | Refractory hyperkalaemia (CAUTION) | KDIGO 2D |
| Insulin/Metformin | NODAT management | KDIGO 1A |

## 3. Acute vs Chronic Management
| Phase | Intervention | Goal |
|---|---|---|
| Acute (high trough + Cr rise) | Reduce CNI 25-50% | Target tac 4-6, Cr improve in days |
| Chronic (biopsy-proven) | Minimise/convert | Preserve GFR, slow progression |
| Conversion | Belatacept (preferred) or mTOR inhibitor | CNI-free or minimised |
| Advanced (eGFR <30, IFTA >50%) | Supportive CKD care | Prepare for RRT |
