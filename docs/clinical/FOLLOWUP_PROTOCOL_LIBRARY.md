# Follow-up Protocol Library

## Disease-Specific Follow-up Protocols (KDIGO 2024)

### 1. IgA Nephropathy (IgAN)

| Parameter | Value |
|-----------|-------|
| Base Interval | 90 days |
| High-Risk Interval | 45 days |
| Visit Schedule | Months 1, 3, 6, 9, 12, 18, 24; Years 3, 4, 5 |
| Required Labs | Creatinine, eGFR, SPOT UPCR, 24h Urine Protein |
| Interval Labs | CBC (q6mo), LFT (q6mo), Albumin (q6mo) |
| Drug Monitoring | RAASi (Cr/K+ 2w post-change, then q3mo), SGLT2i (eGFR q3mo), Finerenone (Cr/K+ wk4) |
| Escalation Triggers | eGFR decline >30% in 3mo, proteinuria >1 g/d despite optimized RAASi, nephrotic-range, RPGN with crescents |
| Discharge Criteria | No proteinuria for 2y, eGFR >60 for 3y, off immunosuppression for 2y |

### 2. Minimal Change Disease (MCD)

| Parameter | Value |
|-----------|-------|
| Base Interval | 90 days |
| High-Risk Interval | 45 days |
| Visit Schedule | Weeks 1, 2; Months 1, 2, 3, 6, 9, 12, 18, 24 |
| Required Labs | Creatinine, eGFR, SPOT UPCR, Albumin |
| Interval Labs | CBC (q3mo), LFT (q3mo), FBG (q6mo), HbA1c (q6mo) |
| Drug Monitoring | Steroid (CBC, glucose q30d), MMF (CBC, LFT q30d), CNI (CBC, Cr, trough q30d), Rituximab (CBC, CD19 q3mo) |
| Escalation Triggers | Steroid-dependent relapsing, frequent relapses (>2/y), CNI toxicity, infection on immunosuppression |
| Discharge Criteria | Complete remission for 2y, off immunosuppression for 1y, no relapses for 2y |

### 5. Lupus Nephritis (LN)

| Parameter | Value |
|-----------|-------|
| Base Interval | 60 days |
| High-Risk Interval | 30 days |
| Visit Schedule | Week 2; Months 1, 2, 3, 4, 6, 9, 12, 15, 18, 24; Year 3 |
| Required Labs | Creatinine, eGFR, SPOT UPCR, 24h Urine Protein (high), Albumin, C3, C4, Anti-dsDNA |
| Interval Labs | CBC (q30d), LFT (q3mo), C3 (q3mo), C4 (q3mo), Anti-dsDNA (q3mo) |
| Drug Monitoring | HCQ (annual eye exam after 5y), Steroid (CBC, glucose, HbA1c q30d), MMF (CBC, LFT q30d first 6mo), Cyclophosphamide (CBC, LFT, UA q14d), Rituximab (CBC, CD19, immunoglobulins q3mo), CNI (Cr, trough q30d), Azathioprine (CBC, LFT, TPMT q30d) |
| Escalation Triggers | Rising anti-dsDNA with falling C3/C4, worsening proteinuria >1g/d increase, eGFR decline >25%, active urinary sediment (RBC casts), extra-renal lupus flare |
| Discharge Criteria | Complete renal remission for 3y, off immunosuppression for 2y, stable on HCQ alone, no extra-renal activity for 2y |

### 6. ANCA Vasculitis

| Discharge Criteria | Sustained remission for 3y, off immunosuppression for 2y, ANCA negative for 1y, no relapse during steroid taper |

## General Protocol (Unclassified)

Used when no specific GN diagnosis is available from ClinicalProfile differential or biopsy.

| Parameter | Value |
|-----------|-------|
| Base Interval | 180 days |
| Required Labs | Creatinine, eGFR, 24h Urine Protein |
| Interval Labs | CBC (q6mo), LFT (q6mo) |
| Escalation Triggers | eGFR decline >30% in 3mo, new nephrotic-range proteinuria, AKI hospitalization |
| Discharge Criteria | eGFR >60 for 2y, no proteinuria for 1y, transferred to general CKD clinic |
